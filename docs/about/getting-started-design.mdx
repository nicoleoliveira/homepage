# Design

Este guia tem finalidade de orientar sobre o uso e utilização dos arquivos de design.

## Figma Community File Design Kits

Os arquivo de fundamentos e componentes do [Animalia Design System](https://www.figma.com/@animaliads) estão disponíveis na comunidade do figma por meio do nosso [UI Kit](https://www.figma.com/@animaliads). Esses arquivos podem ser duplicados para sua própria conta do Figma.
Use nossos componentes diretamente ou salve o arquivo como sua própria biblioteca Figma para uso contínuo. Leia mais do Figma sobre como usar arquivos de comunidade. Saiba mais no [Figma](https://help.figma.com/hc/en-us/articles/360038510873) sobre uso dos arquivos.
